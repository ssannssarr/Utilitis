# UTILITIS

A collection of scripts and utilities that I use on a daily basis to make my life easier.

Each script is placed in its own dedicated folder and includes its own `README.md` file whenever needed.

## Installation

```bash
curl https://raw.githubusercontent.com/ssannssarr/Utilitis/refs/heads/main/install.sh | sh 
```
It will automatically clone the repo and install the package globally. You can see the script at [install.sh](./install.sh).
